Module type Sorting.Swap

module type Swap = sig .. end
A Swap module defines the datapath of the sorting network.

The functors which implement the specific sorting network arrange values of type t with an appropriate set of compare_swap operations to create the sorted outputs

type t
type of data to be sorted
val compare_swap : Sorting.dirn ->
       t -> t -> t * t
The main sorting operation. compare_swap Up a b will return (min a b, max a b). compare_swap Down a b will sort the opposite way.
val delay : t -> t
hardware implementation only delays inserted appropriately into the datapath so each stage can be pipelined
