HardCaml examples framework

Overview



The hardware design examples all provide an interface using the Design module type provided by this module.

Each hardware module will provide a description of itself, the parameters required to configure it and functions to instantiate the hardware and run a testbench.

The framework will collect configuration parameters from the user, create the hardware design and run some user defined tasks such as RTL generation, simulation etc.

Implementations of the framework suitable for a command line application or javascript based webpage will be provided.
