module Hierarchy : sig ... end
Generate circuit with hierarchy
val write : ?transforms:Transform.transform_fn list -> Circuit.Hierarchy.database -> string -> (string -> Circuit.t -> unit) -> Circuit.t -> unit
write ~transforms ~database circuit_name write_module circuit recursively scans circuit and finds all sub-circuits (ie instantiations). These are looked up database and write_module is called for each sub-circuit.
Appropriately designed circuits can thus be split over multiple layers of hierarchy and files.
