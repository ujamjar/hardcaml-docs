Module Vg

Declarative 2D vector graphics.
Vg is a declarative 2D vector graphics library. In Vg, images are Vg.image that denote functions mapping points of the cartesian plane to colors. The library provides Vg.I to define and compose them. Renderers for Vgr_pdf, Vgr_svg and the HTML Vgr_htmlc are distributed with the library. An API allows to implement new renderers.
Consult the basics, the semantics and examples.
